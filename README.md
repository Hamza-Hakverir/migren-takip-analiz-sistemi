# Migren Takip ve Analiz Sistemi

Bu proje, migren hastalarının ataklarını kayıt altına alması, tetikleyicilerini analiz etmesi ve yapay zeka destekli risk tahminleri alması için geliştirilmiş web tabanlı bir sağlık uygulamasıdır.

## Projenin Amacı
Migren atakları karmaşık ve kişiye özel süreçlerdir. Bu uygulamanın temel hedefleri şunlardır:
* Atakların detaylı (5 aşamalı) form yapısı ile kayıt altına alınması.
* Beslenme, hava durumu ve duygu durumu gibi tetikleyicilerin grafiksel analizi.
* Geliştirilen algoritma ile anlık migren risk tahminlemesi yapılması.

## Kullanılan Teknolojiler
Proje geliştirilirken aşağıdaki teknolojiler ve kütüphaneler kullanılmıştır:

* **Backend:** Python (Flask Framework)
* **Veritabanı:** SQLite3
* **Frontend:** HTML5, CSS3 (Modern Dark UI), JavaScript
* **Veri Görselleştirme:** Chart.js
